问题：360 谷歌 浏览器的兼容性

情况：点击左侧列表，右侧无法出现内容页

原因：jquery 的 .load() 方法无效

解决方法1：
    关闭浏览器
    开始 -> 运行 -> 输入
    360chrome.exe --allow-file-access-from-files

解决方法2：
    关闭浏览器
    浏览器快捷方式 -> 右键 -> 属性 -> 快捷方式选项卡 -> 目标
    在 xxxx.exe 后面加入 --allow-file-access-from-files
