• 分支结构
  • 【单项分支】 if(变量==值){

    多条PHP语句 }

    【双项分支】 if(变量==值){

    多条PHP语句 }else{

    多条PHP语句 }

    		★ else前的区间为真区间
    ★ else后的区间为假区间
    【多项分支】 if(变量==值){

    多条PHP语句 }elseif{

    多条PHP语句 }else{

    多条PHP语句 }

    【多项分支】 switch(变量==值){

    case 值:多条PHP语句;break;

    case 值:多条PHP语句;break;

    ...

    default:多条PHP语句 }

    		★ case 值必须是确定值 不能是范围值
    ★ 不能使用浮点型
    ★ break 用于破坏执行进程 当符合条件时不再往下读取
    ★ default 相当于 else
    【巢状分支】 if(变量==值){

    多条PHP语句

    if(变量==值){

    多条PHP语句

    if...

    }

    else{

    多条PHP语句

    } }else{

    多条PHP语句 }
