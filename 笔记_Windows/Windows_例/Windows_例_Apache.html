配置目标

确保所有开发运行库已安装


配置

配置文件

配置文件路径

/conf/httpd.conf

配置 Apache 基本路径常量

Apache 的安装路径
ServerRoot "e:/apache"

配置 web 服务器路径

项目文件所在路径（如果加载了虚拟主机需要将虚拟主机也做相同配置）
DocumentRoot "e:/WWW"

配置服务器名

将 servername 的注释打开并修改
ServerName localhost:80

配置 PHP 解析功能

让 Apache 支持 PHP
在 #LoadModule 列表下添加
#php for apache in module
LoadModule php5_module "e:/php/php5apache2_4.dll"
PHPIniDir "e:/php"
AddType application/x-httpd-php .php
