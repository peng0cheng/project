【判断】【if】

if( 条件 ){ 条件为 true 时执行的代码 }
【判断】【if...else】

if( 条件 ){ 条件为 true 时执行的代码 }else{ 条件不为 true 时执行的代码 }
【判断】【if...else if...else】

if( 条件1 ){ 条件1为 true 时执行的代码 }else if( 条件2 ){ 条件2为 true 时执行的代码 }else{ 不满足以上所有条件时执行的代码 }
【判断】【switch...case】

switch( 变量 ){ case 值1 : 变量等于值1时执行的代码 case 值2 : 变量等于值2时执行的代码 default : 不满足以上所有条件时执行的代码 }
【循环】【while】

while( 循环条件 ){ 每次循环执行的代码 }
【循环】【do...while】

do{ 每次循环执行的代码 }while( 循环条件 )
【循环】【for】

for( 循环开始前执行 ; 循环条件 ; 循环结束后执行 ){ 被执行的代码块 }
【循环】【for...in】

for( 变量 in 对象 ){ 被执行的代码块 }
【控制】【break】

break
【控制】【continue】

continue
【错误异常处理】【try...catch】

try{ js代码 }catch( 变量 ){ 处理错误 }
【错误异常处理】【throw】

throw 自定义错误信息
